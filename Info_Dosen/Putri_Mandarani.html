Research Profile


My research is integrated in mobile technology. The purpose of my research is to produce a system that occurs interaction between
hardware and software by applying the concept Internet of Things (IoT). The system works automatically and implements the
mobile concept. The sensor network becomes a solution in an automated system. In my research, some of the issues that can be
covered include (1). Mobile based application. In this case, i study design, implementation and test apps. The apps used to
monitoring or controlling hardware in automation based system (2). Data communication process using mobile network
and security. In this topic, i study the fundamental computer network and apply cryptography concept or combination (3).
Embedded system in smart technology. In this topic, i study how analog to digital or digital to analog conversion process based
network sensor (4) Augmented Reality (AR). This topic is focused for question "how device mobile can capture marker?" And then
study built apps for promotion based AR and user satisfaction. Overall, all the courses I teach based on information technology have
contributed to my research.
